# Global-Fertility-Trends-Dashboard

## Project Overview
This project presents an interactive data visualization story that tracks and analyzes global fertility and demographic trends over several decades. By visualizing complex population dynamics, the dashboard aims to provide insights into regional shifts, economic correlations, and future demographic projections for policymakers and researchers.

## Tech Stack & Tools
- **Tableau**: The primary tool used for data visualization and storytelling, allowing for a highly interactive user experience.
- **Data Sourcing**: Leveraged datasets from organizations such as the World Bank and United Nations Population Division.
- **Data Preparation**: Cleaned and structured demographic data to ensure consistency across varying global reporting standards.

## Methodology
The development of the data story followed a user-centric design approach:
1. **Trend Identification**: Analyzing historical fertility rates to identify significant global and regional inflection points.
2. **Storyboarding**: Planning the narrative flow of the visualization to guide the user through key demographic transitions.
3. **Dashboard Construction**: Building interactive maps, trend lines, and heatmaps in Tableau to represent diverse demographic metrics.
4. **Insight Extraction**: Identifying correlations between fertility rates and socioeconomic factors such as female education and GDP per capita.
5. **Interactive Optimization**: Refining filters and parameters to allow users to explore trends based on specific regions, income levels, or time periods.
